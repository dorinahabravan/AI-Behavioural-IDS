# AI Behavioural Intrusion Detection Dashboard

## Overview

This project presents an AI-powered behavioural intrusion detection dashboard developed for cybersecurity risk prediction using network traffic analysis.

The system analyses network traffic behaviour and predicts malicious activity using multiple machine learning models trained on the CICIDS2017 dataset.

The dashboard was developed as part of a BSc (Hons) Computing dissertation project.

---

## Features

- Real-time traffic simulation
- Behavioural anomaly detection
- Machine Learning intrusion detection
- Multiple AI models:
  - Random Forest
  - Logistic Regression
  - Isolation Forest
  - 1D CNN
- SHAP explainability integration
- Live attack monitoring
- Interactive Streamlit dashboard
- Attack logging system

---

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- TensorFlow/Keras
- SHAP
- Plotly
- Pandas
- NumPy

---

## Dataset

The project uses the CICIDS2017 dataset for behavioural network traffic analysis and intrusion detection research.
